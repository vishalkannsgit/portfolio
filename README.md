This portfolio is a single self-contained `index.html` — no framework, no build step, no dependencies.

| Feature | Description |
|---|---|
| 🍎 Apple-inspired design | Clean white theme, Inter font, minimal UI |
| 🖱️ Custom cursor | Dot + trailing ring with hover morph effect |
| ✨ Cursor trail particles | Subtle blue particles that follow mouse movement |
| 🧲 Magnetic buttons | Buttons pull toward cursor on hover |
| 🃏 3D card tilt | Project/skill cards tilt based on mouse position |
